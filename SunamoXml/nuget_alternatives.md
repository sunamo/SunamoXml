# NuGet Alternatives to SunamoXml

This document lists popular NuGet packages that provide similar functionality to SunamoXml.

## Overview

XML utilities

## Alternative Packages

### System.Xml.Linq
- **NuGet**: System.Xml.Linq
- **Purpose**: LINQ to XML
- **Key Features**: XDocument, XElement, modern XML API

### System.Xml
- **NuGet**: System.Xml
- **Purpose**: Built-in XML
- **Key Features**: XmlDocument, XmlReader, XmlWriter

### System.Xml.Serialization
- **NuGet**: System.Xml.XmlSerializer
- **Purpose**: XML serialization
- **Key Features**: Object to XML mapping

### XmlDocument
- **NuGet**: System.Xml.XmlDocument
- **Purpose**: DOM-based XML
- **Key Features**: DOM manipulation

## Comparison Notes

System.Xml.Linq (LINQ to XML) is the modern approach. XmlSerializer for object mapping.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Xml.Linq**: LINQ to XML
- **System.Xml**: Built-in XML
- **System.Xml.Serialization**: XML serialization
